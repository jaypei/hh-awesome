
# About

Useful awesome wm config framework.

# Dependency

* compton
* rofi
* shutter
* slock
* alsa-utils


# Usage

1. Install dependencies
1. Setup dotfiles

        cd ~/.config
        git clone --recursive https://github.com/jaypei/hh-awesome.git awesome

1. Restart your awesome window manager

        echo "awesome.restart()" | awesome-client
